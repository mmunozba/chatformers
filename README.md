# chatformers
Lean, model-agnostic framework for text generation models in the Huggingface transformers library. Includes a chat GUI and API server.

## Status
| Feature | Status |
| --- | --- |
| REST API | ❌ |
| Chat Frontend | ❌ |
| Transformers Model Support | ❌ |

## Running locally

### Prerequisites
1. Install Python.
2. Run
`cd backend/api`
`pip install -r requirements.txt`
